# data/raw/

This folder is committed empty in your fresh clone. After running the
pipeline you should see:

- `arxiv_raw_metadata.csv` — produced by `src/01_fetch_arxiv_data.py`
- `Data_Set_S1.txt` — the raw labMT 1.0 file from the supplementary
  materials of Dodds et al. (2011), placed here manually. See the
  README §7.2 for the source.
